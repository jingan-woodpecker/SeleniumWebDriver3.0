系统管理命令： ps -aux

    ps命令可以查看进程的详细进程
    
    选项                         含义
     -a                          显示终端上所有进程，包括其他用户的进程
     -u                          显示进程的详细状态
     -x                          显示没有控制终端的进程
     -w                          显示加宽，以便显示更多的信息
     -r                          只显示正在运行的进程
